Material Design Lite

O Material Design Lite é uma página de design que fornece templates, ícones e códigos para estilização e criação de componentes.

✅ Templates:

Como o nome já diz essa é uma opção em que o MDL disponibiliza templates responsivos para download.

✅ Coomponents:

Os componentes são criados com HTML, CSS e JavaScript, esses componentes podem ser usados para compor páginas web e aplicativos, deixando eles mais atraentes.

Para usar o Slider basta entrar no site da MDL e copiar os seguintes códigos: 

                    
                        <input class="mdl-slider mdl-js-slider" type="range"
                        min="0" max="100" value="0" tabindex="0">
e também inserir as tags <link> e o <script> no HEAD. 
                    
                        <script src="https://code.getmdl.io/1.3.0/material.min.js"></script>
                        <link rel="stylesheet" href="https://code.getmdl.io/1.3.0/material.indigo-pink.min.css">
                        <link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">

✅ Styles:

Na aba de estilos a página fornece ícones dos tipos mais variados para download em SVG e PNG, e também é possível alterar a cores dos ícones.
